# PetDesk 开发路线图

> 按顺序逐个完成，每步有可验证的产出。渲染引擎已调整为**视频素材方案**。

---

## 第一阶段：骨架 — 视频渲染验证

### Step 1 — Electron + Vite + 透明视频播放 ✅
- [x] 初始化 Electron + Vite + TypeScript
- [x] 无边框透明置顶窗口 + 系统托盘
- [x] IPC 通信（前台窗口查询 / 显示隐藏）
- [ ] 替换 Canvas 占位宠物 → `<video>` 播放 WebM Alpha 视频
- [ ] 准备一段测试用 WebM Alpha 视频（idle 循环）
- [ ] 验证透明视频在 Electron 窗口中的透明效果
- **✅ 验证**：一只半透明背景的真实感宠物视频在桌面右下角循环播放

### Step 2 — 视频切换引擎
- [ ] 实现 VideoController：预加载视频列表 → src 切换 → crossfade 过渡
- [ ] 至少 3 段测试视频：idle / happy / walk
- [ ] 切换逻辑：loop: true 循环，loop: false 播完自动回 idle
- [ ] 预加载策略：首屏 idle 立即加载，其余后台队列加载
- **✅ 验证**：按键 1/2/3 无缝切换 idle → happy → walk，过渡无闪烁

### Step 3 — Canvas 叠加层
- [ ] Canvas 覆盖在 `<video>` 上方，透明背景，pointer-events: none
- [ ] 绘制文字气泡（圆角矩形 + 文字）
- [ ] 绘制简单粒子（雨滴 / 星星）
- [ ] 气泡显示/隐藏动画（fadeIn + 3s 后 fadeOut）
- **✅ 验证**：点宠物 → 视频切换 + Canvas 弹出气泡"你好呀~"

## 第三阶段：互动 — 让宠物有反应

### Step 7 — 全局鼠标监听
- [ ] 主进程全局鼠标钩子，50ms 采样坐标
- [ ] 鼠标状态机：IDLE / HOVER_PET / DRAGGING / IDLE_LONG
- **✅ 验证**：控制台打印鼠标状态变化

### Step 8 — 鼠标交互行为（7 种）
- [ ] 悬停 ≥1s → 播放 attentive 视频
- [ ] 单击 → happy 视频 + 气泡 + 心情 +2
- [ ] 快速拖动 → scared 视频 + 躲角落
- [ ] 静置 ≥3min → attention 视频（扒光标）
- [ ] 双击 → 随机特殊视频
- [ ] 右键 → 菜单（表情包/设置/切换素材）
- [ ] 拖动文件经过 → chase 视频
- **✅ 验证**：7 种交互全部触发

### Step 9 — 剪贴板监听 + 桌面拖动
- [ ] 剪贴板文本长度 >200 → surprised 视频
- [ ] 用户可拖拽宠物改变位置
- [ ] 拖到屏幕边缘自动吸附
- [ ] 30s 无拖动恢复窗口跟随
- **✅ 验证**：复制大段文字 → 惊跳；拖拽贴边 → 吸附
